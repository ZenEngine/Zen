## 使い方

1. このリポジトリをクローンします。

```
git clone https://github.com/ZenEngine/Zen.git
```

2. Visual StudioからCMakeプロジェクトとして読み込みます。

## Requirement

### Windows

- Visual Studio 2022
- CMake

### Linux

```
apt-get install autoconf
```
